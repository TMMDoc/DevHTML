Trace Gatherer

To activate Brick, following lines should be available in Middleware.ini (found in C:\ProgramData\Metamation\RA\RData folder).

[Basic]
....
[Experimental]
....
[Brick]
Enabled=1

All the required files can be found in this link.

  • Copy StartApp.dll and StartApp.pdb into C:\Program Files\Metamation\RA\RightAngle folder.

  • Copy startapp-settings.json to C:\ProgramData\Metamation\RA\RData folder.

  • Copy TraceGatherer folder to C:\Program Files\Metamation\RA. After copying the above files and folder, there will be C:\Program Files\Metamation\RA\TraceGatherer\TraceGatherer.exe executable file.

  • Settings related to trace can be edited in C:\Program Files\Metamation\RA\TraceGatherer\settings.json.
