Translation tables

Translation tables are managed in Flux using online XLate tool.

This can be found here.

Refer this for usage of this online translation tool.

To keep the translations updated, a developer has to:

  • Update the remote translation database with the new phrases from a new release which have to be translated.

  • Pull the remote translation updates and synchronize the local dictionaries. This will update the .txt files in X:\Wad\Lang.

Steps

  • Refer this for installing XL tool and the commands to use.

  • After installing, do CLI sign-in.

  • Now, switch to the product Flux as shown below:

    xl sw Flux "X:\Wad\lang" "Y:\Xlate\0"

  • To upload the translation phrases that have to be translated:

    1. Use xl out to see the changes going to the remote server

      xl out Flux -diff

    2. Use xl put command to seed the translation phrases from the local translation dictionaries

      xl put Flux "X:\Wad\lang" "Y:\Xlate\0"

  • To download the translated texts for the phrases uploaded earlier:

    1. Use xl in and xl recent commands to check the incoming and recently updated translations

      xl in Flux

    2. Use the xl get command to download and apply the remote changes to the local dictionaries

      xl get Flux
