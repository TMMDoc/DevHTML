Machine ID Scheme

This document explains the scheme we use to assign machine IDs for Flux machines. We try to follow a more rational scheme than ahs been done in MetaCAM, so that inspection of the IDs can give us some clue about the machine. The machine ID has the following form:

MMTTNNN

  • MM is the manufacturer code. This can be from 1 .. 3 digits long, with the smaller codes reserved for the more major players.

  • TT is the <i>technology</i> code (like laser, bend, combination etc). This is always exactly 2 digist long

  • NNN is the index within that manufacturer/technology cluster. This is always exactly 3 digits long
