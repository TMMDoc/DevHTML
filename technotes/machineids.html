Machine ID Scheme

Machine ID Scheme

This document explains the scheme we use to assign machine IDs for Flux machines. We try to follow a more rational scheme than ahs been done in MetaCAM, so that inspection of the IDs can give us some clue about the machine. The machine ID has the following form:

MMTTNNN

  • MM is the manufacturer code. This can be from 1 .. 3 digits long, with the smaller codes reserved for the more major players.

  • TT is the <i>technology</i> code (like laser, bend, combination etc). This is always exactly 2 digits long

  • NNN is the index within that manufacturer/technology cluster. This is always exactly 3 digits long

Machine IDs (Trumpf-Custom)

This document provides a complete list of all the Machine IDs assigned for custom Trumpf machines. These are special one-off machines that are specific to particular customers. They are included as part of the Flux Machines.Wad file, but do not show up for normal installation in the Machine installation dialog.

Installing a custom machine

These custom machines cannot be user-installed - they do not show up in the machine list. To install one of these machines, use these steps.

1.Start a command prompt, and navigate to the Flux Data folder. Depending on the type of Flux installation, this will vary. For example, it is ProgramData\Metamation\Flux for a Metamation branded Flux installation, or ProgramData\Trumpf\Flux for a TecZoneBend stand-alone installation. 2.Go to the Machine subfolder inside the Data folder. 3.Create a subfolder in there with the machine ID. If the machine ID is 320001, then create a folder called 320001 inside the Machine folder. 4.Create a file inside that folder called marker.bin - the contents of this file are not important, and the file can be an empty file. Here is an example after we have followed all these steps machine 320001: Restart Flux after this, and the machine can now be used. It can also be configured from the Machine Configuration dialog, though it cannot be removed. (We prevent removal, since if the user removed this machine by mistake, they have no way to get it back).
