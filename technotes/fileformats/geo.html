.GEO File Format

.GEO File Format

Excerpt from https://trumpf.atlassian.net/wiki/spaces/TTBC/pages/103583626/GEO-Format

The GEO file is split into blocks and sections.

  • #~1 : Drawing or single part

  • #~11 : Information about the drawing

  • #~3 : General part data

  • #~30 : Description texts for a part (as of T300 V3.2, T100 V5.0)

  • #~31 : Point data of a part/subpart

  • #~32 : Elements of a part/subpart/pattern

  • #~33 : Contour data of a part/subpart/pattern

  • #~34 : Data of a part copy, with description text between #~TEXTINFO and ###~TEXTINFO

  • #~331 : Elements assigned to a contour

  • #~332 : Equidistant elements of a contour

  • #~35 : Data of a subpart

  • #~36 : Element attributes (since V2.2)

  • #~37 : Information about a bend in the part/subpart

  • #~38 : Bend attributes (since V2.3)

  • #~371 : Bending lines

General terminators:

  • End of file: #~EOF

  • End of block: #~END, #~KONT_END, #~BIEG_END, #~SUB_END, #~TTINFO_END

  • End of section: ##~~

  • End of element/attribute: |~

Data Description

Drawing Data

++#++~1 // Drawing data
Database version // 1.03 if new Info data, 1.02, 1.01
Revision number
Date created or revised
Minimum values (x y z)
Maximum values (x y z)
Surface area
Data unit // 1 = MM, 2 = INCH
Accuracy
2D or 3D model // 0 = 2D, 1 = 3D
Number of parts
++##++~~

Drawing Information

++#++~11 // Information about the drawing
Drawing number
Drawing name
Customer
Programmer
Job number or name
Material designation
Sheet thickness

Machine name

Number of columns for block layout (since 1.03)
Number of rows for block layout (since 1.03)
Code for machining method (laser/punching) (since 1.03)
Code, if part in Mini-Nest/component (since 1.03)
Name of Mini-Nest or component (since 1.03)
++##++~~
++#++~END

Part Text Information (Block #~30)

Text information about the part supports the following keyword meanings:

  • AUFTR : job

  • BEARB : programmer

  • BEZCH : part name

  • DREHINKR : increment for rotation

  • IDENT : part id

  • MASCH : machine

  • MAT : material

  • PART_NESTFITTYPE : lay part in a grid

  • PART_VERSION : format version

  • PRIOFRGEO : nesting priority

  • RGLWK : rules

  • ROTALLOWED : part may be rotated

  • SOURCE_FILE_NAME : original file

  • TABS : tables

  • TKUND : customer entry

  • TMUSTER : is sample part needed

  • TruMasName : machine name

  • WERKSTOFF : raw material

Further keywords can be added.

++#++~30
Keyword1@Value1 // Text information about this part
Keyword2@Value2
++#++~TTINFO_END

Point Data

++#++~31 // The points should be sorted according to X value.
Point number
++|++~

++|++~
++#++~BEND_ATTRIBUTE_END

Elements (Block #~32)

Legend:

  • Color: 0 = black, 1 = white, 2 = red, 3 = yellow, 4 = green, 5 = cyan, 6 = blue, 7 = magenta

  • Line type: 0 = solid, 1 = dashed, 2 = dotted, 3 = dashed-doted

  • Point symbol: 0 = point, 1 = plus sign, 2 = asterisk, 3 = circle, 4 = cross, 5 = triangle, 6 = square, 7 = rhombus

++#++~32
PKT // Point element
Point symbol color
Point number
Number of element attributes (since V2.2) // if number = 0, the whole block can be skipped
Attribute number 1 (since V2.2)
Attribute number 2 (since V2.2)
++|++~

LIN // Line element
Color linetype
Startnr Endnr
Number of element attributes (since V2.2) // if number = 0, the whole block can be skipped

++|++~

Color linetype

Text element notes:

  • Text type: 1 = ISO, 130 = ISO prop, 131 = bold

  • Text centering: 9 = Bottom left, 17 = Bottom center, 33 = Bottom right, 10 = Center left, 18 = Center, 34 = Center right, 12 = Top left, 20 = Top center, 36 = Top right

  • Text direction: 1 = left-to-right, 4 = up-to-down

Contour Data

Contour notes:

  • Contour type: 24 = Closed contour, 25 = Open contour

  • Contour position: 0 = Outside contour, 1 = Inside contour

  • Contour direction: (0,0,1) = Counter-clockwise, (0,0,-1) = Clockwise

Bend Data

Bend notes:

  • Bend type: 0 = Normal bending, 1 = Hem, 2 = Radius bend, 4 = Flat processing, 5 = Z-bend, 7 = Double sheet bending, 8 = swing folding

  • Method: 0 = Air bending, 1 = Coining, 2 = seam with gap, 3 = flattening, 4 = swing folding

  • Technique: 0 = Normal bending, 1 = Pre-/Final bending, 10 = Imprinting, 11 = Joining, 12 = Forming, 13 = Punching
