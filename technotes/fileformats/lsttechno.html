.LST file - technology

In the program sections 'SP' and 'HP' technology commands are used

TC_LASER_ON

Definition

With the TC_LASER_ON cycle call-up, the following machine commands are executed independently of the entered cycle parameters:

  • Switch on the distance regulation.

  • Select piercing gas.

  • Move Z axis to piercing position.

  • Switch on beam for piercing.

  • Select the cutting gas.

  • Move Z axis to cutting position.

  • Switch on beam for cutting.

  • Select focal position.

  • Select beam diameter.

  • Select laser power.

Programming

TC_LASER_ON (Laser method,"Technology table",piercing type,cutting type)

  • The parameters of the technology table for TC_LASER_ON for different materials and material thicknesses are available at the machine control. The name of the suitable technology table must be specified at the time of programming.

  • If pure cutting or piercing cycles are called up, the value "0" must be entered for the cycle parameter "Piercing type" or "Cutting type".

  • Is deleted by TC_LASER_OFF.

Laser method

The following laser methods are available:

Laser method Functions

1 or 9

Piercing and cutting with distance regulation.

2 or 10

Piercing and cutting without distance regulation.

3

Piercing without distance regulation, cutting with distance regulation.

4

Piercing with distance regulation, cutting without distance regulation.

5 or 11

Cutting with distance regulation.

6 or 12

Cutting without distance regulation.

7

Piercing with distance regulation, beam OFF.

8

Piercing without distance regulation, beam OFF.

30

Microweld
Piercing type

Cycle parameters: data is selected from the active technology table via the piercing type. Several piercing types are stored there:

Number Functions

0

No piercing

10

Normal piercing

11

Gentle piercing

20

Dot punch marking

30

Special processing hard Microweld

31

Special processing soft Microweld
Cutting type

Cycle parameters: Data is selected from the active technology table via the cutting type. Cutting data (e. g. laser output, cutting and approach parameters) is stored there for different machining requirements. Differences are as follows:

  • Size, medium or small contour.

  • Normal or reduced cutting speed.

  • Normal or reduced acceleration values.

The cutting type is selected using a three-digit number, whereby each digit has a particular significance:

1st digit: contour 2nd digit: cutting speed 3rd digit: acceleration

1 = large contour

0 = normal

0 = normal

2 = medium contour

1 = reduced from piercing hole

1 = reduced

3 = small contour

-

2 = high

Cutting with "floating data exchange"

With laser methods 5 or 11 and 6 or 12, cutting with "floating data exchange" can be programmed. If a further TC_LASER_ON instance is programmed with methods 5 or 11 and 6 or 12 and the beam has already been ignited, switching over can be done as follows:

  • Switch among the various cutting data within a technology table (e. g. due to a contour change).

  • Switch among the various technology tables.

No axis stop is necessary in case of modifications of laser power and gating frequency as well as for minor modifications of cutting parameters such as gas pressure, gating frequency, and cutting distance. The switch will be made to the new parameters during the machining.

Advantages

  • No pits due to reduction of the cutting speed during data exchange.

  • Faster processing due to "floating data exchange".

TC_LASER_OFF

Definition

Depending on the parameters entered, the following machine commands are executed with the TC_LASER_OFF cycle call-up:

  • Switch off the laser beam.

  • Position the Z axis.

Programming

The TC_LASER_OFF cycle is called up with a cycle parameter in the NC program:
TC_LASER_OFF(Laser method)

Laser method

The following TC_LASER_OFF cycles are available:

Laser method Functions

1

Beam off, Z axis at overshoot height (standard overshoot height = 40 mm).

2

Beam off, cutting gas off, high voltage off, Z axis at home position.

3

Corresponds to method 1.

4

Beam off, Z axis remains at processing clearance (at piercing height for the following piercings / at cutting height for the following cuts).

6

In the case of RotoLas: Laser beam off, Z axis to overshoot height (in relation to the outer circle).

10

Beam off, preparation for hard Microweld.

11

Beam off, preparation for soft Microweld.

The laser methods are used as follows:

  • If Laser method 1 is programmed, the overshoot height to approach the Z axis must first be programmed with TC_POS_LEVEL. This method is used to switch off the laser beam after a processed part and to position the next part.

  • Laser method 2 is used to switch off the laser beam at the end of part programming and to move the Z axis to the home position.

  • Laser method 3 corresponds to Laser method 1.

  • Laser method 4 is used when processing "Common cuts" and for SprintLine. During positioning, there must never be a hole or formed section.

TC_WAIT

Definition

Laser cycle for corner cooling to prevent burning in. The laser beam is switched off.

Programming

TC_WAIT (Function, Time)

Functions Description

No information

Waiting time from the laser technology table is used.

0

Waiting time from the laser technology table is used.

1

Delay time is used from the transfer parameter "Time".

2

Waiting for start button (unconditional stop M0).
Note
 The function is optional.
Functions Description

Time

Time indication in seconds.

G823 (FlyLine)

Definition

Programming a contour-related beam suppression. With the G823 function, the laser beam can be suppressed in relation to the block. At the same time, the beam at the block transition is switched off unsupported. If another travel block without a G823 function follows a travel block with this function, the beam will be ignited at the block transition unsupported. The respective process parameters remained unchanged thereby. At the same time, the beam suppression is not restricted to linear blocks.

Programming

G823

Note
 The G823 function is assigned to a group of G commands. Only one G command may be used from this group within an NC block. The G commands G820 – G829 belong to this group.

  • The function only becomes effective if the laser processing was previously activated by TC_LASER_ON.

  • A piercing process must be possible without axis standstill when the material is selected or the material thickness is selected.

  • Depending on the machine, FlyLine can be used for various material thicknesses (for exact information on the material thickness, see the data collection for the machine concerned).

TC_POS_MODE

Definition

The TC_POS_MODE cycle controls the positioning process of the cutting head between the cutting processes.

Programming

TC_POS_MODE (Value)

  • Self-holding (modal) operation.

  • Is deactivated by the renewed programming of the cycle, the end of the program or the actuation of the Reset button.

Value Explanation

0

Standard behavior.

1

Unsupported retraction of the contour after TC_LASER_OFF.

2

Prerequisite for the programming of this value is that programming be done without piercing and with TC_LASER_OFF(4).

  • Floating approach of the contour with TC_LASER_ON.

  • Unsupported retraction of the contour after TC_LASER_OFF.
Note

  • Optimum positioning paths are urgently required when using the TC_POS_MODE(1). Pay attention especially to possible falling parts and holes in the traversing path of the cutting head.

  • If no TC_LASER_OFF(4) is programmed during the use of TC_POS_MODE(2), then the function G823 is automatically deactivated for the contour start. No error is output.

TC_FLUID_ON (Option)

Definition

The cycle TC_FLUID_ON activates water spraying. Water spraying is used during laser flame cutting.

Programming

TC_FLUID_ON

  • Water spraying is generally controlled via the machine technology tables (program properties). Mode 0 or 1 switches water spraying on or off generally in the overall program. If in the machine technology table mode 2 is active, water spraying can be activated in a program-controlled manner via the cycle TC_FLUID_ON and deactivated via TC_FLUID_OFF.

  • Is deactivated by program end, program interruption or TC_FLUID_OFF.

  • Contour related: must be programmed before TC_LASER_ON.

Definition

The TC_FLUID_OFF cycle deactivates water spraying.

  • Water spraying is generally controlled via the machine technology tables (program properties). Mode 0 or 1 switches water spraying on or off generally in the overall program.   • Deactivate TC_FLUID_ON.

  • Is deactivated by program end or program cancellation.

  • Contour related: must be programmed after TC_LASER_OFF.
