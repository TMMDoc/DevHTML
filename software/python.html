Python

Excerpt from: Python + conda / pip install guide - TW543 Data + AI - Confluence (trumpf.com)

Use the Python version that best suits your needs: https://www.python.org/downloads/windows/.

Ensure to tick the option "Add Python to environment variables" during installation. This allows you to use Python directly from the command prompt.

Note
 This method is not recommended if it requires setting a username and password.

Configuring pip.ini with Proxy Settings

To configure Trumpf’s local Python PyPi repository (jFrog-based) for installing packages, follow these steps:

  1. Install Python on the C: drive.

  2. Navigate to AppData\Roaming\pip under your user directory on the C drive.

  3. Create a pip.ini file with the following content:

    [global]
trusted-host = artifactory.corp.trumpf.com
index = https://pypi.org/
index-url = https://pypi.org/simple

  4. If the pip folder does not exist, create it manually.

Installing Python Packages with pip

To install Python packages using pip, open the command prompt and run the following command:

    python -m pip install numpy

Creating executable Python Scripts

Python 3.9.9 works well with the pyinstaller package to create executable Python scripts
