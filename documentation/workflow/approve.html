Pre-Publish

Merge Pull Request

Once the pull request is approved, it should be merged the main branch.

  1. To merge a pull request, go to the specific GitHub repository → Pull request page.

  2. Click Merge pull request and select Confirm merge.

Ensure to resolve conflicts, if any.

Generate HTML Files

Run Bento to compile AsciiDoc files and generate HTML files.

Ensure you pull the contents from remote main branch to your local main branch.

Following are the steps to generate the HTML files:

  1. From your local Git folder, open Command Prompt and type git br to see list of all branches.

  2. Type git sw main to switch to the main branch.

  3. Type git pull to pull the latest content from the main branch.

  4. Type bento . to generate HTML files.

    1. Once HTML files are generated, commit and push the files to the relevant HTML GitHub repository.

      1. If it is a new document, create a new URL by following the steps in Publish.

      2. If it is an existing document, it will deploy automatically.

Ensure that the Auto-deploy in Render setting is ON.
