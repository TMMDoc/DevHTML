Publish

For Approver only

  1. Sign in into Render using GitHub account.

  1. Click New + and select Static Site to create a static website.

  1. Connect a GitHub repository that you wish to deploy.

    1. Search for <TitleBookHTML> repo.

  1. Fill in the fields in the following format:

Name = <TitleBookHTML>

Branch = main

Root directory = ./

Build command =

Publish directory = .

  1. Click Create Static site.

  2. Wait for few seconds for the Render to host your site.

    1. You should see the site Live with latest commit message.

  1. Click the url to do a final review.

The URL will be created only once per document and remains the same.

  1. Goto, Render Dashboard and select the <TitleBookHTML>.

  1. Navigate to Settings → Build & Deploy → Auto-Deploy and edit it to No.
