Installation

Installation

  1. Install MS Visual Studio Code

  2. Add AsciiDoc Extension

  3. Install Ruby Gems for AsciiDoctor

  4. Download Bento

MS Visual Studio Code

  • Contact IT administrator to install the latest version of MS Visual Studio Code.

AsciiDoc Extension from VS Code

  • Open VS Code application and click the Extensions icon.

  • On the search field, type asciidoc and click Install under the AsciiDoc extension.

AsciiDoc Syntaxes used in documentation

Refer to AsciiDoc Syntax Quick Reference

Install Ruby Gem

  1. Go to ruby installer and download the set up file.

  2. Go to downloads, unblock the setup file and run rubyinstaller.

  3. Select Only for me when prompted.

  4. At the end of the installation click the Finish.

  5. Ensure a command prompt window opens with MSYS2 installed properly message.

Install AsciiDoctor

  1. Open command prompt and type gem install asciidoctor to install asciidoctor.

  2. Type asciidoctor --version command to confirm the asciidoctor installation.

  3. Type command gem install asciidoctor-html5s to create html files.

  4. Type gem install asciidoctor-diagram.

Set Environment Variables

Follow the steps to set Environment Variable for successful installation of Ruby gem and Bento.

Ruby gem

  1. Open command prompt and type set path.

  2. Locate for Ruby installation folder.

  3. Add C:\Ruby33-x64\bin to path in user variable.

Install and Run Bento

  1. Place Bento.exe file in C:\Work.

  2. Add C:\Work to path variable.

  3. Type bento in command prompt to check for compatibility.

  4. Open command prompt and go to any of the document folder (Book).

  5. Type bento . to compile.

  6. Check for errors, fix broken links and remove orphaned images, if any.
