Tools

Every content developer should strive to have the best structured documentation.

Restructuring files, folders and images in Bento documentation is a little tricky. Here are some script tools to restructure. They are not self explanatory and mostly based on matlab scripts.

Wolf Wadehn will help to restructure the Table of Contents.

Convert Word to Bento (Word2Bento.m)

A script that cleans up a via Pandoc-generated adoc file for use in Bento.

C:\Users\wadehnwo\AppData\Local\Pandoc\pandoc Vulcan.docx -f docx -t asciidoc --wrap=none --markdown-headings=atx --extract-media=extracted-media -o output.adoc

Convert Filestruc to Btxt (Bentofilestruc2txt.m)

A script which converts a Bento file structure into a text file like:

.\contents.txt
  .\index.adoc
  .\fds.adoc
  .\8_Windowssystemcontrol\contents.txt
  .\8_Windowssystemcontrol\index.adoc

Convert Btxt to Mindmap (Btxt2MM.m)

A script which converts the Btext file into a mind map for graphical rearrangement. Use Freemind.com.

Convert Mindmap to Btxt (MM2Btxt.m)

A script which converts the Btext again out of MM.

Convert Btxt to a new Bento project (MM2Btxt.m)

Creates a new file structure in the subfolder "new" and sorts all pics in the right directories.
