We use Github as a source code repository, FogBugz as an issue tracking system, and Render for static website hosting. Installers for products are hosted on OneDrive / GoogleDrive etc.

Fogbugz Projects

Each team at Metamation has its own Fogbugz project(s) and areas. In addition:

  • The Tools project is used to report bugs and request features for the tools we maintain.
