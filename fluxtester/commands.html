Command Cookbook

This cookbook summarizes the command patterns

Quick Command Surfaces

  • RunScript("…​"): execute Flux API commands without returning a value.

  • RunScriptWithOutput("…​"): execute commands and capture result string output.

  • RunScript(List<string>): execute a multi-line command sequence.

  • Type("…​") and Type(KeyShort.XXX): trigger keyboard-driven commands in the UI.

Keyboard Command Cheatsheet

Common command keys used throughout the file:

  • Type("b"): switch/trigger bend workflow or bend tech actions.

  • Type("c"): trigger post-process / NC generation style command path.

  • Type("w"): open or use workflow panel actions.

  • Type("1"), Type("3"): numeric workflow responses in bend prompts.

  • Type(KeyShort.TAB): commit edited field values.

  • Type(KeyShort.ESC): close/cancel active UI mode.

  • Type(VirtualKeyShort.F11): toggle fullscreen in one test flow.
