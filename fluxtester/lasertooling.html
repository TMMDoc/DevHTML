  • C122144 - Changing the pierce condition (Normal, Gentle, Mark, None)

  • C122145 - Changing the approach parameters (dimple approach, approach length/radius/angle variations)

  • C122162 - Changing the withdrawal/escape parameters (escape length, radius, angle)

  • C150731 - Adding microjoint to remove scrap tooling

  • C123179 - Adding and deleting scrap tooling

  • C150730 - Adding and managing microjoints (0 to 5 microjoints)

  • C122142 - Opening and closing laser cut edit dialog

  • C168079 - Cut/Mark open polyline backwards (forward vs. backward direction)

  • C200388 - Process for open polylines (None, Mark, Cut, Based on layer)

  • C200390 - Allow approach that is more than 0.5 distance to opposite side

  • C200394 - Corner processing with cool settings where rounding/looping not possible

  • C200395 - Scrap cutting with approach length for separating cut

  • C200398 - Scrap cutting with scrap grid width settings

  • C145702 - Auto-recognize countersink curves and assign forming footprint layer

  • C122167 - Creating and deleting microjoints (with NC code generation)

  • C145687 - Skip point entities setting (None, Points on polylines, All)

  • C145682 - Laser cut glob “To point operation” conversion (path ↔ point)

  • C145706 - Display “mark forming footprint” setting in Laser CAM cutting section

  • C145732 - Pie strategy for cutting circular slugs

  • C145734 - Delete pie cut when scrap tooling is created

  • C3526 - Recompute tooling after enabling dimple on pie strategy sample

  • C202419 - Adding corner treatment processes (Looping, Dwell, Rounding, Stop, Cool, Slowdown, Do Nothing)

  • C214095 - Mark bend lines (with 3 processes: None, Partial, Full; valid & invalid bend data)
