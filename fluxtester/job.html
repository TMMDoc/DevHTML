Job.cs Overview

Purpose

Job.cs is a comprehensive test fixture for validating Job and Layout management functionality in the Flux cutting application. It tests job creation, part management, sheet handling, and advanced nesting operations with priority, rotation, and grouping.

Fixture Details

Property Value

Fixture ID

1500

Name

“Nesting Job Tests”

Module

FluxCut

Subsystem

NestJob

Test Machines

2230001, 97

Primary File

123.fx

Pattern

IDisposable (auto-cleanup)

Setup & Teardown

Constructor (IDisposable Pattern)

public Job () {
    Utils.ConfigureMachine (TESTMACHINEID1);  // Machine 2230001
    Utils.ConfigureMachine (TESTMACHINEID2);  // Machine 97
}

public void Dispose () {
    Utils.UninstallMachine (TESTMACHINEID1);
    Utils.UninstallMachine (TESTMACHINEID2);
    Utils.DeleteTempFile ();
}

Key: Uses IDisposable pattern instead of [FixtureInitialize]/[FixtureCleanUp] for more flexible resource management.

Test Methods (24 Tests)

Layout Management

C150732 (Test 401) - “Adding and deleting layouts”

  • Opens job file (TwinJob.fxjob)

  • Tests adding 6 layouts (1 → 7 total)

  • Tests deleting 3 layouts (7 → 4 total)

  • Tests deleting middle layout (4 → 3 total)

  • Validates layout counts

C150733 (Test 402) - “Moving layouts”

  • Adds layout to job

  • Tests move up operation

  • Tests move down operation

  • Compares automation tree dumps with reference files

Job Creation & Dialog Management

C122130 (Test 1528) - “Selecting ‘New Job’ toolbar button opens a pop-up dialog”

  • Opens Flux application

  • Clicks “new job” button

  • Verifies NewJobDlg appears

C122132 (Test 1529) - “In ‘New Job’ dialog without adding mandatory fields clicking ‘Cancel’ should close the dialog”

  • Opens Flux and New Job dialog

  • Tests Cancel button functionality

  • Verifies dialog closes

Part Management

C200408 (Test 1530) - “Save Parts(s) from the job”

  • Creates new job

  • Adds 2 parts (123.fx, InnerContour.fx)

  • Saves parts to temp directory

  • Verifies file existence

  • Cleans up saved files

C122176 (Test 1531) - “Add and delete parts in the job”

  • Creates new job

  • Adds multiple parts

  • Verifies parts appear in parts table

  • Tests deleting parts

  • Validates deletion

C122179 (Test 1532) - “Increase the quantity of the part”

  • Opens job (TwinJob.fxjob)

  • Adds extra parts

  • Modifies part quantities via textbox

  • Handles material warning dialog

  • Renests with new quantities

  • Validates parts count in layout

Sheet Management

C150396 (Test 1534) - “Add and delete sheets in the sheets tab of a new job”

  • Creates new job

  • Adds 5 sheets (1 → 6 total)

  • Verifies 6 sheets with size “3000x1500”

  • Deletes all 6 sheets

  • Validates deletion

C162420 (Test 1535) - “Export remainder sheet”

  • Opens job (TwinJob.fxjob)

  • Creates remainder sheet

  • Opens export dialog

  • Exports to DXF file

  • Compares with reference file

  • Cleans up temporary file

C162522 (Test 1536) - “Import remainder sheet”

  • Opens job (TwinJob.fxjob)

  • Checks initial parts count (5)

  • Deletes existing sheet

  • Imports remainder sheet from DXF

  • Verifies imported sheet dimensions (1460.85x1500)

  • Renests and validates parts count (4)

Part Properties & Configuration

C122180 (Test 3527) - “Set the priority for the part”

  • Opens job (NestingJobSample.fxjob)

  • Tests 4 priority combinations: [1,2,3], [2,1,3], [3,2,1], [0,1,2]

  • Sets priorities via parts table

  • Renests after each priority change

  • Validates layout with priority arrangement

  • Generates multiple layouts

C122181 (Test 3528) - “Set the part rotation”

  • Opens job (AsymmetricalPolygon.fxjob)

  • Tests 6 rotation options

  • Handles material warning dialog

  • Sets rotation via context menu

  • Renests after each rotation change

  • Validates layout with different rotations

C122182 (Test 3529) - “Group different parts”

  • Opens job (NestingJobSample.fxjob)

  • Tests 4 group combinations: [1,2,3], [2,1,3], [3,2,1], [0,1,2]

  • Sets group IDs via parts table

  • Renests after each grouping

  • Validates layouts with different groupings

Copy/Paste/Cut Operations

C145731 (Test 3530) - “Cut/Copy & Paste parts in layout sheets”

  • Opens job (AsymmetricalPolygon.fxjob)

  • Tests Copy/Paste workflow

    • Copies part (Ctrl+C)

  • Tests Cut/Paste workflow

    • Cuts part (Ctrl+X)

    • Pastes part (Ctrl+V)

    • Validates parts within sheet margin

    • Compares layout

Key Patterns Used

IDisposable Pattern

public class Job : IDisposable {
    public Job () {
        // Setup in constructor
        Utils.ConfigureMachine (...);
    }

    public void Dispose () {
        // Cleanup
        Utils.UninstallMachine (...);
    }
}

Advantage: Automatic cleanup via using statement or test framework

Architecture Highlights

  • IDisposable Pattern: Automatic cleanup on test completion

  • Modular Helpers: Reusable methods for common operations

  • External Validation: Uses Nesting class for layout validation

  • Script Integration: Heavy use of Flux scripting for validation

  • UI Automation: Comprehensive FlaUI usage for WPF controls

  • Reference-Based Testing: Compares against baseline files

  • Multi-scenario Testing: Tests various combinations (priorities, rotations, groups)

  • Dialog Management: Handles multiple dialog types and warnings
